# Projet-IA-Fueli
De l'anglais fuel — le carburant. Fueli, c'est l'idée que la nourriture est le carburant du corps : ce que tu manges détermine ce que tu peux accomplir. Le nom est court, mémorable, et ancré dans une vision concrète de l'alimentation — pas de restriction, pas de régime, juste comprendre ce que tu mets dans ton corps.
